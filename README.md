# Webサイトテンプレート

- Next.js
- Tailwind
- GSAP
- Lenis
- Page Transition Animation
- Storybook

## ローカルで実行

1. Install dependencies:

```sh
pnpm i
```

2. Start the dev server:

```sh
pnpm dev
```
